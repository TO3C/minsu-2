# 流云智炬 - 三亚湾精品民宿网站

一个现代化的民宿展示网站，支持中文、英文、俄文三语切换，采用清新的三亚海滩风格设计。

## 快速开始

```bash
cd my-app
npm install
npm run dev
```

访问 http://localhost:3000

## 项目文档

详细文档请查看 [my-app/README.md](my-app/README.md)

## 部署

1. 将此代码推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 "GitHub Actions"
4. 推送代码后自动部署

## 项目结构

```
.
├── .github/workflows/     # GitHub Actions 配置
├── my-app/               # Next.js 项目
│   ├── src/              # 源代码
│   ├── dist/             # 构建输出
│   └── README.md         # 详细文档
└── README.md             # 本文件
```

## 技术栈

- Next.js 14 + TypeScript
- Tailwind CSS
- GitHub Pages（托管）
- Formspree（表单服务）
